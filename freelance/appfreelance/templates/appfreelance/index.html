Información

- HTML (HyperText Markup Language): Es el lenguaje fundamental para crear la estructura de las páginas web.

- CSS (Cascading Style Sheets): Se utiliza para diseñar y dar estilo a los elementos HTML, como colores, tipografías, márgenes, etc.

- JavaScript: Es un lenguaje de programación que se utiliza para agregar interactividad a las páginas web, como animaciones, validaciones de formularios, efectos visuales, entre otros.

- Frameworks y Bibliotecas**: Tales como React.js, Angular, Vue.js para JavaScript, y Bootstrap, Foundation para CSS, proporcionan herramientas y estructuras predefinidas para agilizar el desarrollo web.
